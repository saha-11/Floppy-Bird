# FLOPPY BIRD

A colorful Flappy Bird style game built with HTML, CSS and JavaScript.

Fly through pipes, beat your high score, switch themes, and survive gravity.

---

## ✨ Features

🌤️ Light Mode  
🌈 Gay Mode (rainbow visuals + custom sounds)  
🐦 Animated bird with movement effects  
🎵 Different sound effects for jumping, scoring and crashing  
☁️ Moving background elements  
🏆 High score saved automatically  
💬 Random funny messages while playing  
📱 Works on desktop and mobile  

---

## 🎮 Controls

### Desktop
SPACE → Jump  
Mouse Click → Jump  

### Mobile
Tap screen → Jump  

---

## 🏆 Objective

Pass through pipes and get the highest score possible.

Avoid:
- Pipes
- The ground
- Embarrassing crashes
